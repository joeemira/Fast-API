# 📘 FastAPI Posts API

A simple RESTful API built with **FastAPI** and **PostgreSQL** to perform CRUD operations on blog posts.

---

## 🧠 Features

- ✅ Fetch all posts
- 📌 Retrieve a specific post by ID
- ✍️ Create a new post
- ✏️ Update a post
- ❌ Delete a post
